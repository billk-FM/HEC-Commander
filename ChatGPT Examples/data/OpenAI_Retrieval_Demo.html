Default Settings for ChatGPT Assistants (GPT's)

  • Chunk size: 800 tokens
  • Chunk overlap: 400 tokens
  • Max results: 20 for GPT-4, 5 for GPT-3.5-turbo
  • Token budget: 16,000 for GPT-4, 4,000 for GPT-3.5-turbo

Notes:

  1. Adjacency percentage will alter number of chunks that overlap, reducing the effective token budget and total number of non-overlapping tokens returned. 100% adjacency will produce 8000 effective non-overlapping tokens with the default 50% overlap.
  2. Default values represent the default behavior of ChatGPT GPT's.
  3. General relevancy is the assumed number of tokens that would be relevant enough for the user to expect them to be returned to the model.
